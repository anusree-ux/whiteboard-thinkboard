# 🧠 ThinkBoard – Real-time Collaborative Whiteboard

A powerful, minimal, real-time collaborative whiteboard built using **React**, **Next.js**, **Fabric.js**, and **Socket.IO**. ThinkBoard helps teams brainstorm, draw, and communicate visually from anywhere.

## 🚀 Features

- 🎨 Draw with pencil and eraser tools
- 🖊️ Add text, images, and shapes
- 🧭 Pointer tool for object selection and movement
- 🧑‍🤝‍🧑 Real-time multi-user collaboration via Socket.IO
- 🔍 Zoom & pan support
- 🌈 Dark mode and theme toggling
- 💬 In-app chat sidebar
- 📷 Export whiteboard as image
- 📌 Join or create rooms with unique IDs
